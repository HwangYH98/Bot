/맛집검색 <가격> :

유저들이 사용하는 명령어 입니다.
<가격>란에는 원하시는 가격대를 입력합니다.

원하시는 가격을 입력하시면 이용자별, 카테고리별, 목적별, 분위기별로 확인 할 수 있는 버튼이 뜹니다.
이용자별 : 20대, 30대, 40대, 50대, 60대이상, 뒤로가기
카테고리별 : 배달, 밥집, 카페, 술집, 고깃집, 횟집, 한식, 중식, 일식, 양식, 아시안, 멕시칸, 이탈리안, 뷔폐, 브런치, 패스트푸드, 분식, 국물요리, 면요리, 해산물, 뒤로가기
목적별 : 아침식사, 점심식사, 저녁식사, 혼밥, 혼술, 혼카페, 데이트, 회식, 건강식, 다이어트, 가족외식, 아이동반, 실버푸드, 식사모임, 술모임, 차모임, 접대, 간식, 뒤로가기
분위기별 : 가성비좋은, 분위기좋은, 푸짐한, 격식있는, 고급스러운, 서민적인, 시끌벅적한, 조용한, 깔끔한, 이색적인, 뷰가좋은, 예쁜, 지역주민이찾는, 뒤로가기

뒤로가기는 공통으로 있으며, 클릭 시 이용자별, 카테고리별, 목적별, 분위기별 버튼이 있는 임베드를 다시 띄웁니다.

이후 각 범주별로 원하시는 버튼을 누를 경우 해당하는 맛집을 5개씩 끊어서 보여줍니다.
추천 맛집이 5개 이상일 경우 더보기 버튼이 생기며, 더보기를 누를 경우 추가로 5개의 추천 가게를 띄우며, 만약 그럼에도 5개 이상의 추천가게가 남아있다면 더보기 버튼을 생성합니다.
각 임베드당 보여주는 내용은 다음과 같습니다.

가게 이름

주소 : 하이퍼링크로 클릭시 네이버지도로 연결됩니다.
전화번호 : 가게의 전화번호를 보여줍니다. 만약 게시되지 않은 경우 "게시되지 않음"으로 표시됩니다.
영업시간 : 가게의 영업시간을 보여줍니다. 만약 게시되지 않은 경우 "게시되지 않음"으로 표시됩니다.
메뉴 : 가게의 메뉴와 가격을 보여줍니다. 
평점 : 가게의 평점을 보여줍니다. 만약 게시되지 않은 경우 "게시되지 않음"으로 표시됩니다.
바로가기 : 하이퍼링크로 클릭시 다이닝코드 내 해당 가게의 정보 홈페이지와 연결됩니다.
